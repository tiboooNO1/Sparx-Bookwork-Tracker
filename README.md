<h2>🛠️ Installation Steps:</h2>

<p>1. Chromium Browsers</p>

```
Go to the extensions page and enable "Developer mode". Click "Load unpacked" select the uncompressed folder you downloaded from the releases page and you're done!
```

<p>2. Firefox Browsers</p>

```
Go to "about:config" search for "xpinstall.signatures.required" and set its value to "false". Next open your extensions page click the "gear icon" select "Install Add-on From File..." choose the ".xpi" file you downloaded from the releases page and you're done!
```
